Welcome to NearDrop

How would you like to use this device?

Host
Share files from this machine. Other devices on your network connect to you.
  • Runs a local file server
  • Generates connection PIN
  • Full file management
  • Other devices browse your files
Client
Connect to another NearDrop host. Browse and transfer files from their machine.
  • No server needed — lightweight
  • Enter host URL to connect
  • Full read & write access
  • Upload & download files
