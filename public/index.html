HOW TO USE NEARDROP
Host Mode
Your device becomes a file server. Other devices on the same network can connect to you, browse your shared folder, upload files, and download files. You're the central hub.
Client Mode
Your device connects to a host. You enter the host's URL and PIN to access their shared files. No server runs on your machine — you're just browsing and transferring.
Quick Start
1. Make sure both devices are on the same WiFi network
2. On one device, launch NearDrop in Host Mode
3. On the other device, open NearDrop in Client Mode — or scan the QR code / open the URL in any browser
4. Enter the PIN shown on the host to connect
5. Drag & drop files to transfer in either direction!
Tips
• Use the Chat sidebar to send text snippets between devices
• You can switch between Host and Client mode from Settings
• Multiple devices can connect to a single host simultaneously
• No internet required — everything stays on your local network
